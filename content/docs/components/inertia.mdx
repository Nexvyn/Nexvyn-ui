# Inertia Cards

An interactive card component that uses GSAP's InertiaPlugin to create smooth, physics-based animations. Cards respond to mouse movement with momentum and rotation effects based on real-time velocity tracking.

## Features

- **Physics-based inertia** - Cards animate with realistic momentum calculations
- **Mouse velocity tracking** - Real-time delta X/Y tracking for smooth interactions
- **Random rotation inertia** - Cards spin with varied momentum on hover
- **Smooth easing** - GSAP InertiaPlugin for natural motion
- **Responsive grid layout** - Adaptive card grid
- **GPU optimized** - Uses `will-change-transform` for performance

## Installation

This component requires GSAP and the InertiaPlugin:

```bash
npm install gsap @gsap/react
```

## Basic Usage

<Preview>
  <InertiaCards />
</Preview>

```tsx
import { useEffect, useRef } from "react";
import { gsap } from "gsap";
import { InertiaPlugin } from "gsap/InertiaPlugin";
import { useGSAP } from "@gsap/react";

gsap.registerPlugin(InertiaPlugin);

const images = [
  "https://i.pinimg.com/736x/63/5b/b8/635bb894b77cf88c78873aa43f02150e.jpg",
  "https://i.pinimg.com/736x/1b/15/f2/1b15f21748612520d08c90c1b883627e.jpg",
  "https://i.pinimg.com/736x/5e/31/d0/5e31d02ecfe0fed9c4e7767ebb86fe77.jpg",
  "https://i.pinimg.com/736x/1d/66/96/1d669647743a55264389471098adc3bf.jpg",
  "https://i.pinimg.com/736x/f4/3c/9f/f43c9f0f5bf4a70df07fa31d307ba48e.jpg",
  "https://i.pinimg.com/1200x/ed/14/71/ed14717f99c678ffd75ba46df2245aa9.jpg",
  "https://i.pinimg.com/736x/87/b2/d5/87b2d5a8653c7d06d7f3b5df5ad75bbf.jpg",
  "https://i.pinimg.com/736x/69/25/2a/69252a688a59bcd684e43eb06a5629da.jpg",
  "https://i.pinimg.com/736x/2a/eb/7c/2aeb7cbaeb168fe023d36a5b13485e9c.jpg",
];

const InertiaCards = () => {
  const deltaX = useRef(0);
  const deltaY = useRef(0);

  useEffect(() => {
    let oldX = 0;
    let oldY = 0;

    const handleUpdateDelta = (e: any) => {
      deltaX.current = e.clientX - oldX;
      deltaY.current = e.clientY - oldY;

      oldX = e.clientX;
      oldY = e.clientY;
    };

    window.addEventListener("mousemove", handleUpdateDelta);

    return () => {
      window.removeEventListener("mousemove", handleUpdateDelta);
    };
  }, []);

  useGSAP(() => {
    const cards = gsap.utils.toArray(".cards");

    const handleMouseEnter = (e) => {
      const tl = gsap.timeline();
      tl.to(e.currentTarget, {
        rotation: 0,
        inertia: {
          x: {
            velocity: deltaX.current * 20,
            end: 0,
          },
          y: {
            velocity: deltaY.current * 20,
            end: 0,
          },
          rotation: { velocity: (Math.random() - 0.5) * 300, end: 0 },
        },
      });
    };

    cards.forEach((card: any) => {
      card.addEventListener("mouseenter", (e: any) => {
        handleMouseEnter(e);
      });
    });

    return () => {
      cards.forEach((card: any) => {
        card.removeEventListener("mouseenter", handleMouseEnter);
      });
    };
  });

  return (
    <div className="flex justify-center items-center h-screen w-full overflow-hidden">
      <div className="grid grid-cols-3 gap-4">
        {images.map((src, index) => (
          <img
            key={index}
            className="cards w-52 h-52 rounded-3xl object-cover object-top will-change-transform"
            src={src}
            alt={`Inertia Card ${index}`}
          />
        ))}
      </div>
    </div>
  );
};

export default InertiaCards;
```

## How It Works

### Velocity Tracking

The component tracks mouse movement to calculate velocity:

1. **Mouse Event Listener** - Listens to `mousemove` events globally
2. **Delta Calculation** - Calculates the difference between old and new positions
3. **Velocity Storage** - Stores X and Y velocity in refs for access on hover

### Inertia Animation

When hovering over a card, GSAP's InertiaPlugin calculates momentum:

1. **Capture Velocity** - Uses stored `deltaX` and `deltaY` values
2. **Calculate Momentum** - Multiplies velocity by 20 for pronounced effect
3. **Smooth Deceleration** - InertiaPlugin naturally decelerates to end value
4. **Rotation Inertia** - Random rotation velocity for varied animation

### Physics Properties

| Property            | Value                         | Purpose                         |
| ------------------- | ----------------------------- | ------------------------------- |
| velocity (X)        | `deltaX.current * 20`         | Horizontal momentum             |
| velocity (Y)        | `deltaY.current * 20`         | Vertical momentum               |
| velocity (rotation) | `(Math.random() - 0.5) * 300` | Random spin effect              |
| end (X/Y/rotation)  | `0`                           | All animations settle to center |

## Customization

### Adjusting Velocity Multiplier

Change the momentum intensity by modifying the velocity multiplier:

```tsx
x: {
  velocity: deltaX.current * 50,  // More intense
  end: 0,
},
y: {
  velocity: deltaY.current * 50,  // More intense
  end: 0,
},
```

### Changing Grid Layout

Modify the grid columns and gap:

```tsx
<div className="grid grid-cols-4 gap-6">
  {" "}
  {/* 4 columns instead of 3 */}
  {images.map((src, index) => (
    <img
      key={index}
      className="cards w-56 h-56 rounded-3xl object-cover object-top will-change-transform"
      src={src}
      alt={`Inertia Card ${index}`}
    />
  ))}
</div>
```

### Customizing Card Dimensions

Update the width and height classes:

```tsx
className =
  "cards w-64 h-64 rounded-3xl object-cover object-top will-change-transform";
```

### Adding/Removing Images

Extend or reduce the images array:

```tsx
const images = [
  "image-url-1.jpg",
  "image-url-2.jpg",
  "image-url-3.jpg",
  // Add more or remove existing
];
```

### Adjusting Rotation Range

Modify the random rotation velocity:

```tsx
rotation: {
  velocity: (Math.random() - 0.5) * 500,  // Wider rotation
  end: 0
}
```

Main card component with inertia physics.

### Inertia Configuration

| Property          | Type   | Value                 | Description                    |
| ----------------- | ------ | --------------------- | ------------------------------ |
| x.velocity        | number | deltaX \* 20          | Horizontal momentum from mouse |
| y.velocity        | number | deltaY \* 20          | Vertical momentum from mouse   |
| rotation.velocity | number | (random - 0.5) \* 300 | Spin momentum                  |
| x.end             | number | 0                     | Final X position (center)      |
| y.end             | number | 0                     | Final Y position (center)      |
| rotation.end      | number | 0                     | Final rotation angle           |

## Performance Optimization

### GPU Acceleration

The component uses `will-change-transform` for GPU optimization:

```tsx
className = "cards w-52 h-52 rounded-3xl object-cover will-change-transform";
```

### Event Delegation

All cards share a single event listener setup to reduce memory usage:

```tsx
cards.forEach((card: any) => {
  card.addEventListener("mouseenter", (e: any) => {
    handleMouseEnter(e);
  });
});
```

### Cleanup

Proper cleanup in useGSAP return function prevents memory leaks:

```tsx
return () => {
  cards.forEach((card: any) => {
    card.removeEventListener("mouseenter", handleMouseEnter);
  });
};
```
