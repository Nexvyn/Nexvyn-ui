# Button

A feature-rich button component that supports multiple variants, sizes, visual effects, and icons.

## Basic Usage

<Preview>
  <Button>Click me</Button>
</Preview>

```tsx
import { Button } from '@/components/Button';

export default function Example() {
  return <Button>Click me</Button>;
}
```

## Icon Support

Add icons to your buttons with `icon` and `iconPlacement` props:

<Preview>
  <div className="flex flex-wrap gap-2">
    <Button>
      <span className="mr-2">‚Üê</span>
      Icon Left
    </Button>
    <Button>
      Icon Right
      <span className="ml-2">‚Üí</span>
    </Button>
  </div>
</Preview>

```tsx
import { ArrowLeft, ArrowRight } from 'lucide-react';

<Button icon={ArrowLeft} iconPlacement="left">
  Previous
</Button>

<Button icon={ArrowRight} iconPlacement="right">
  Next
</Button>
```

### Expand Icon Effect

The `expandIcon` effect makes icons expand into view on hover:

```tsx
<Button effect="expandIcon" icon={ArrowRight} iconPlacement="right">
  Hover to See Icon
</Button>
```

## Advanced Usage

### As Child (Polymorphic)

Use `asChild` to render the button as a different element:

```tsx
import Link from 'next/link';

<Button asChild>
  <Link href="/docs">View Docs</Link>
</Button>
```

### Combining Multiple Props

```tsx
<Button 
  variant="outline" 
  size="lg" 
  effect="ringHover"
>
  Enhanced Button
</Button>
```
