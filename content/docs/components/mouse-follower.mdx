## Features

- **Image trail effect** - Creates beautiful image trails following cursor
- **Vortex removal** - Unique implosion animation when images disappear
- **Randomized positioning** - Each image spawns with random rotation
- **Performance optimized** - Uses refs and efficient DOM manipulation

## Customization

### Changing Images

Replace the images array with your own:

```tsx
const images = ["/your-image-1.jpg", "/your-image-2.jpg", "/your-image-3.jpg"]
```

### Adjusting Trail Behavior

Modify the config object:

```tsx
const config = {
  imageLifespan: 600, // How long images stay visible (ms)
  mouseThreshold: 40, // Minimum mouse movement to spawn image
  inDuration: 600, // Fade-in animation duration
  outDuration: 800, // Fade-out animation duration
  baseImageSize: 120, // Default image size
  minImageSize: 60, // Minimum random size
  maxImageSize: 120, // Maximum random size
}
```

### Container Size

Adjust the container dimensions:

```tsx
<div className="h-96 w-96" ref={containerRef}></div>
// Change to:
<div className="h-screen w-full" ref={containerRef}></div>
```
