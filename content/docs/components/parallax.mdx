## Features

- **Multi-layered parallax** - Different scroll speeds for depth effect
- **Smooth scroll synchronization** - Uses ScrollTrigger for precise timing
- **Responsive animations** - Works seamlessly across different screen sizes
- **Performance optimized** - Uses GSAP's efficient rendering pipeline

## Customization

### Adjusting Parallax Speed

Modify the `y` values to change how far each layer moves:

```tsx
// Faster parallax (more movement)
tl.to(i1.current, { y: -300, ease: "power2" }, 0)
tl.to(i2.current, { y: -200, ease: "power2" }, 0)

// Slower parallax (subtle movement)
tl.to(i1.current, { y: -100, ease: "power2" }, 0)
tl.to(i2.current, { y: -75, ease: "power2" }, 0)
```

### Changing Easing

Use different GSAP easing functions:

```tsx
// Smooth easing
tl.to(i1.current, { y: -220, ease: "power3.inOut" }, 0)

// Constant speed
gsap.from(img.current, { y: -120, ease: "linear" })
```

### Adjusting Scroll Trigger Range

```tsx
scrollTrigger: {
  trigger: containerRef.current,
  start: "center center",
  end: "center -200",
  scrub: 2,
}
```
