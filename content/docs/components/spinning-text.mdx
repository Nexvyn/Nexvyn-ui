```

```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string \| string[]` | Required | Text to display in circular layout |
| `duration` | `number` | `10` | Duration of one full rotation in seconds |
| `reverse` | `boolean` | `false` | Rotate counter-clockwise |
| `radius` | `number` | `5` | Radius of the text circle (in ch units) |
| `transition` | `Transition` | - | Framer Motion transition config |
| `variants` | `object` | - | Custom animation variants |
| `className` | `string` | - | Additional CSS classes |

## Features

- **Smooth rotation** - Powered by Framer Motion for silky animations
- **Customizable radius** - Control the size of the text circle
- **Reversible direction** - Spin clockwise or counter-clockwise
- **Accessible** - Includes screen reader text

## Examples

### Slower Rotation

```tsx
<SpinningText duration={15}>
  view • more • info •
</SpinningText>
```

### Reverse Direction

```tsx
<SpinningText reverse>
  loading • please • wait •
</SpinningText>
```

### Larger Radius

```tsx
<SpinningText radius={8} className="text-2xl">
  nexvyn ui components •
</SpinningText>
```
