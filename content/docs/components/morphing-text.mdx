## Usage

```tsx
import { MorphingText } from "@/components/ui/morphing-text"
```

```tsx
<MorphingText texts={["Hello", "World", "Nexvyn"]} />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `texts` | `string[]` | Required | Array of words to cycle through |
| `className` | `string` | - | Additional CSS classes |

## Features

- **Smooth transitions** - Uses blur and opacity for seamless morphing
- **SVG filters** - Leverages SVG color matrix for crisp text edges
- **Performance optimized** - Uses requestAnimationFrame for smooth animations
- **Customizable timing** - Configurable morph and cooldown durations

## Examples

### With Custom Styling

```tsx
<MorphingText
  texts={["Design", "Build", "Ship"]}
  className="text-6xl text-primary"
/>
```

### Multiple Words

```tsx
<MorphingText
  texts={["React", "Next.js", "TypeScript", "Tailwind"]}
/>
```
