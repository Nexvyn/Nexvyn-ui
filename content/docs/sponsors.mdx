# Support & Sponsorship

Thank you for using Pixel Perfect. Your support helps us continue building high-quality, motion-driven UI components.

---

## About This Project

Pixel Perfect is developed and maintained by **Aman (Nexvyn)** and **Vansh**. We work to provide production-ready animated components powered by Motion and GSAP.

---

## How to Support

### 1. Star on GitHub
Star our [repository](https://github.com/Nexvyn/ui.git) to help other developers discover the library.

### 2. Share the Library
Recommend Pixel Perfect to other developers or share it on social media and dev communities.

### 3. Contribute Code
Submit bug fixes, improve documentation, or add new components. Visit our [GitHub repository](https://github.com/Nexvyn/ui.git) to contribute.

- **GitHub**: [github.com/Nexvyn/ui](https://github.com/Nexvyn/ui.git)
